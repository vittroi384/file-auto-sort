================================================
   파일 자동 정리 프로그램 — 사용 안내
================================================

파일 이름을 보고, 알맞은 폴더로 자동으로 정리해 주는 프로그램입니다.
(설치 필요 없음. 윈도우용)

※ 중요 — 시작 전에 딱 두 가지만!
   1) 받은 ZIP은 '압축을 완전히 푼 뒤' 사용하세요. (압축 안에서 바로 실행하면 안 돼요.)
   2) 풀어 나온 파일들은 '바탕화면'이나 '문서' 같은 평범한 폴더에 '함께' 두세요.
      (Program Files 같은 보호된 폴더에 두면 설정·기록이 저장되지 않아요.)
   ※ 위치가 잘못되면 프로그램이 켜질 때 "저장할 수 없는 위치" 라고 알려줍니다.


■ 1) 시작하기
------------------------------------------------
1. 받은 ZIP 압축을 '완전히' 풉니다.
   - 풀어서 나온 파일들은 항상 같은 폴더에 함께 두세요.
   - 그 폴더는 '바탕화면'이나 '문서'처럼 평범한 곳에 두세요.
     (Program Files 등 보호된 폴더 X)

2. '파일자동정리.exe' 를 더블클릭합니다.

3. "Windows의 PC 보호" 파란 경고가 뜨면:
      → [추가 정보] → [실행]  을 누르세요.
   (인터넷에서 받은 프로그램이라 뜨는 정상 안내입니다.)


■ 2) 처음 켜면 '설정 창'이 뜹니다  ← 여기만 하면 끝!
------------------------------------------------
경로를 몰라도 됩니다. 버튼으로 폴더만 고르면 돼요.

[1. 정리할 폴더]  ← 여기에 넣는 파일이 자동 정리됩니다.
   - 아래 '빠른 추가'에서  [다운로드] [문서] [바탕화면]  중 원하는 걸 누르거나,
   - [＋ 폴더 추가(직접 고르기)] 를 눌러 원하는 폴더를 클릭해서 고르세요.

[2. 보낼 곳]  ← (선택) 안 써도 됩니다.
   - 정리함에 넣은 파일을, 여기 폴더들 안에 '이미 있는 같은 이름 폴더'로 보냅니다.
   - 필요 없으면 비워두세요.

[저장하고 닫기] 를 누르면 → 작은 '안내 창'이 한 번 더 떠요:
   - 이 프로그램이 '화면 오른쪽 아래 시계 옆'에 숨어서 일한다는 안내예요.
   - 처음에는 '연습 모드'로 시작합니다. (실제로 옮기지 않고 '어디로 갈지'만 보여줌)
     파일을 몇 개 넣어 확인해 본 뒤, 마음에 들면 [바로 정리 시작] 을 누르세요.
     바로 시작하고 싶으면 처음부터 [바로 정리 시작] 을 눌러도 됩니다.


■ 3) 사용하는 법
------------------------------------------------
- '정리할 폴더'로 고른 곳에 파일을 넣으면, 이름을 보고 알맞은
  하위 폴더로 자동으로 들어갑니다.
- 딱 맞는 폴더가 없으면, 소리와 함께 작은 창이 떠서
  폴더 이름을 정하거나 정리를 건너뛸 수 있습니다.
- (연습 모드일 때는 실제로 옮기지 않고, 어디로 갈지만 알림으로 보여줘요.)


■ 나중에 설정을 바꾸려면
------------------------------------------------
화면 오른쪽 아래 시계 옆(숨어 있으면 '^' 클릭)의
'노란 폴더 아이콘' 을 마우스로:
   - 왼쪽 클릭  → 설정 창 열기
   - 오른쪽 클릭 → 메뉴 (일시정지, 연습 모드, 되돌리기, 종료 등)


■ 알아두면 좋은 것
------------------------------------------------
- 처음 실행하면 자동으로 '연습 모드'로 시작해요. 실제로 안 옮기고
  "어디로 갈지"만 알려주니, 마음 편히 먼저 확인해 보세요.
  (나중에 켜고 끄려면: 오른쪽 클릭 메뉴 → '연습 모드')
- 같은 이름 파일이 있어도 덮어쓰지 않고 '파일 (1)' 로 보관합니다.
- 잘못 정리됐으면 오른쪽 클릭 메뉴 → '방금 정리 되돌리기'.
- 끄려면 오른쪽 클릭 → 종료.
